# O-M_Data_Acquisition_Accessability-Validation
This project implements an empirical data pipeline to derive multi-parameter operational limits for offshore wind Operations and Maintenance (O&amp;M) vessels. It aims to replace static heuristics (e.g., $H_s &lt; 1.5m$) with dynamic, vessel-aware workability surfaces using AIS, metocean hindcasts, and structural sensor data.
