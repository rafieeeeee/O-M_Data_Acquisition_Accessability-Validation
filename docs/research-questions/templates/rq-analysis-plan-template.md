# RQ Analysis Plan Template

## Research Question

State the exact research question.

## Hypothesis

State the expected relationship or null expectation.

## Accepted Inputs

List accepted input files, reports, and authority documents.

## Excluded Inputs

List excluded inputs and explain why they are out of scope.

## Claim Boundary

State what the analysis may claim and what it must not claim.

## Method

Describe the planned analysis method without embedding results.

## Validation Checks

List required tests, input checks, output checks, and context checks.

## Outputs

List planned report and data-product paths.

## Caveats

List known caveats that must be carried into interpretation.

## Decision Criteria

Define how the final report will decide between possible outcomes.

## Context Files To Update

List context, roadmap, ADR, handoff, or authority docs that must be updated if
the conclusions change project meaning.
